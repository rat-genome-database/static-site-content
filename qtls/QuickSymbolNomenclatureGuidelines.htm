Quick guide to QTL nomenclature

 

A QTL symbol should:

 

·         include:

־         a stem consisting of an abbreviation of the name (usually the first letters of each word in the name).  The QTL name and trait list should be checked before assigning a symbol.

־         the serial number in the QTL name.

 

A QTL name should:

 

·         include:

־         a root name describing the measured trait/subtrait.  Those QTL affecting the same trait should be given the same stem.  The QTL name and trait list should be checked before assigning a name.

־         the designation QTL

־         a serial number, which will be assigned by RGD and Ratmap after submission of the QTL or curation of the literature

 

 

·         be named for the measured trait and not a disease.  However, some historically named QTL carry the name of the disease with which they are associated, and these names are maintained.
