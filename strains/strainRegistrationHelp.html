ILAR laboratory code:

This is the laboratory code assigned by The Institute of Laboratory and Animal Research to each lab or group. Registering a group at ILAR identifies the group as a place where this strain was originated and maintained.

 

Strain symbol

The strain that is being curated is entered here. For example: BN/Cr, SHR.BN-(D13Arb5-Ren)/Ipcv. Please refer to the rules for the strain nomenclature (link to http://rgd.mcw.edu/nomen/rules-for-nomen.shtml. If there are superscript characters in the name, please contact RGD.Data@mcw.edu. if any help is desired in naming a strain please contact RGD.Data@mcw.edu

 

Parent strain:

This is the name of the parent strain from which the strain to be registered is derived. For example: BN, SHR.

 

Strain type:

This describes the category of the strain i.e. Inbred, Related Inbred, Recombinant Inbred, Coisogenic, Congenic, Consomic, Segregating inbred, Conplastic, outbred, mutant, transgenic.Please use only these terms as this column has a fixed vocabulary.

 

Strain origin

Specific information regarding the origin of this strain is entered in this column. This field could have information like the institution, lab, or company where this strain was originated. For example: 'BN/NHsdMcwi is a Brown Norway strain which came from Ss (now a retired ILAR code) to N (NIH) then to Hsd (Harlan Sprague Dawley) and now it is in Mcwi (Medical College of Wisconsin)'.

 

Source:

The name of the lab, department, University and even the company from where this particular strain can be obtained. Please give the complete details including the address if different from above. If a URL has to be entered, please contact RGD.Data@mcw.edu.

 

Reference:

If this strain has been mentioned in any published paper please give that citation here. ( author, journal, vol, page number, or PMID).
