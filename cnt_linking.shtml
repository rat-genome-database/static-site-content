Linking to the Rat Genome Database

Site Links

If you would like to create a link between your site and the Rat Genome Database, please copy the following code to your page: 

 <p><a href='http://rgd.mcw.edu'><b>Rat Genome Database</b></a>

The link should appear as follows: 

Rat Genome Database

Linking to cgi-queried data

If you would like to link to data which has been queried from the database, copy the RGD ID from the viewing form. Paste the RGD ID onto the end of the following url to link directly to the record: 

http://rgd.mcw.edu/query/query.cgi?id= {paste rgd id here}
