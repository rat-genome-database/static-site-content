RGD Definition

"SSLP stands for Simple Sequence Length Polymorphisms, also known as microsatellite DNA. SSLPs are 1-6 simple nucleotide repeat sequences, highly polymorphic in length of repeat among strains and are often used as genetic markers for genotyping."

Most of the SSLP markers available in RGD have been generated for use in genetic mapping experiments and many of these are placed on the various maps stored within RGD.

Finding SSLP information

1. Using the General Search

The general RGD search available at the top of each page is a quick route to finding SSLPs. This will search SSLP symbols and aliases only. (See General RGD Search help for more information).

2. Using the SSLP Search Form

Quick Guide

1. To find a particular SSLP(s), enter the SSLP symbol or alias symbol (wildcard searches can be performed using the contains option)

2. Searches can be limited to any of several criteria

  • find SSLPs at a particular chromosome, map, and map positions
  • find SSLPs associated or not associated with a particular gene
  • find SSLPs having allele data related to a specific rat strain

3. The output format section allows the user to customize the reporting of the search results to sort alphabetically based on symbol, to sort in ascending order based on chromosome, and to sort based on position. The sorting based on position requires selecting a map and chromosome in the limits section.

SSLP Result reports

The SSLP Result report has several sub-report pages and also has batch download capability.

There are several functions provided on the results page that allow the user to easily navigate through detailed information about the records that match the search criteria.

  • Select/unselect all results on current page by clicking on the check box on the header line
  • Select the wanted items by checking the checkbox at the beginning of each record
  • Navigate records by clicking on a page number at the top or the bottom of the results page
  • View all checked items in one page only by clicking on the "View Selected" button at the bottom of the results page
  • View all checked items in excel format by clicking on the "Download Selected" button at the bottom of the results page
  • View all items in excel format by clicking on the "Download All" button at the bottom of the results page
  • View all map data for selected SSLPs by clicking on the "View Selected Map Data" button
  • View all map data for all SSLPs in the results by clicking on the "Download All Map Data" button
  • Go back to result summary page by clicking on "Go Back" button at the bottom of the results page

Related Information/Links

The SSLP report also contains information on objects related to this SSLP within RGD. This includes links to Strain, QTL, Map and Gene reports. There are also a variety of tools available that contain SSLP data:

  • VCMAP - comparative map tool based on rat radiation hybrid map which contains many SSLPs
  • Genome Scanner - enables the selection of polymorphic markers between two sets of strains

The FTP site contains two files related to the SSLP data contained within the database:

  1. SSLPS file - All fundamental SSLP data contained in RGD
  2. SSLPS_ALLELES - Allele sizes for over 4500 SSLPs in 48 rat strains
