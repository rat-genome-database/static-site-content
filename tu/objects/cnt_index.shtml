What is in RGD?

RGD stores information about various biological objects, which are shown below. RGD has other content such as bioinformatic tools which are described elsewhere

The data in RGD is stored in a relational database which allows for data to be related or linked together forming connections between different sets of information, for example the map location of a gene is stored in a separate place in the database (known as a database table) than the gene information itself. This allows the user to reach all the associated data available for an object and also allows the database search tools to be more powerful by including this associated information in the database searches (e.g. "Find all genes mapped on chromosome 4"). You can view the RGD database schema which shows all the information that is stored in the database and the links between the various objects.

ESTs

Maps
[definition] 		genetic and radiation hybrid maps- map length, number of framework and placement markers. ESTs, Genes, SSLPs, Strains
Sequences
[definition] 		currently limited to SSLP cloned and primer sequences, EST clone and primer sequences. ESTs, Genes, SSLPs
